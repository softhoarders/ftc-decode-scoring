Privacy Policy

Unofficial FTC Scorer · Last updated March 2026

Short version: This app does not collect, store, or transmit any personal data to any server or third party. Everything stays on your device.

1. Who We Are

The Unofficial FTC Scorer is a free, open-source tool developed by Team 12560 Soft Hoarders, based in Craiova, Romania. It is designed to help referees and teams track scores during FIRST Tech Challenge DECODE 2025–26 matches. This tool is unofficial and is not affiliated with, endorsed by, or sponsored by FIRST or any official competition body.

2. Data We Do Not Collect

We do not collect any of the following:

  • Personal information (name, email address, location, etc.)
  • Device identifiers or fingerprints
  • Usage analytics or telemetry
  • IP addresses or network metadata
  • Cookies

There are no third-party analytics, tracking scripts, or advertising networks included in this application.

3. Data Stored Locally on Your Device

The app saves match history (match number, team identifiers, phase scores, timestamps, and ranking points) in your browser's localStorage under the key ftc-decode-v3. This data:

  • Never leaves your device
  • Is never sent to any server
  • Is not accessible to Team 12560 or any other party
  • Can be cleared at any time by clearing your browser's site data or localStorage

4. Audio Resource

The match timer sound effect is loaded from a Vercel Blob Storage URL (zul17livsh2m1kvt.public.blob.vercel-storage.com). This is a static audio file fetch; no personal data is sent in that request beyond what your browser includes in any standard HTTP request (e.g., IP address visible to Vercel's CDN). We do not control or receive logs from Vercel for this request.

5. Progressive Web App (PWA)

If you choose to install the app to your home screen, the browser handles the installation process. No additional data is collected or transmitted as a result of installation. A single localStorage flag (ftc-pwa-dismissed) is saved to remember whether you have dismissed the install banner.

6. Children's Privacy

Because we do not collect any personal data, this app is safe for use by people of all ages, including minors participating in FTC competitions.

7. Changes to This Policy

If we ever change this privacy policy, the updated version will be reflected in this file in the public repository. Given that we collect no personal data, significant changes are unlikely.

8. Open Source

The source code for this application is published under the GNU General Public License v3 and is available for inspection at github.com/softhoarders/ftc-decode-scoring. You are welcome to verify these privacy claims by reading the code directly.
